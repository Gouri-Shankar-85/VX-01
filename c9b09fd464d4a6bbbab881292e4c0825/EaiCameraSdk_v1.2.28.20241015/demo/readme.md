#### USAGE

##### * Use unpack_linux_ros.sh to unpack *linux_ros.pkg*, then you will get linux and ros1/ros2 samples.
* If you wanna run linux or ros, unpack *linux_ros.pkg* in linux system by shell *unpack_linux_ros.sh. Some errors may 		occur if you unpack in windows system.

#### 使用方法

##### 使用脚本unpack_linux_ros.sh解压*linux_ros.pkg*包，即可得到 linux/ros/ros2系统的参考例子。

##### 如果想使用linux/ros/ros2系统的参考例子，务必使用脚本unpack_linux_ros.sh去解压*linux_ros.pkg*包。如果在windows系统下解压，可能会出现错误。
